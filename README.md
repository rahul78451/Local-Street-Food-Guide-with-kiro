ğŸ“– Description
==============

Delhi Eats is a web application designed to help users explore the diverse and vibrant street food scene in Delhi. The application provides a comprehensive guide to local street food, including popular dishes, restaurants, and food stalls. With Delhi Eats, users can easily discover new places to eat, read reviews, and get directions to their favorite food spots.

The application is built using Streamlit, a powerful Python library that allows for the creation of web applications with a simple and intuitive interface. Delhi Eats is designed to be user-friendly, with a clean and responsive design that works seamlessly on desktop and mobile devices. The application is perfect for foodies, tourists, and locals alike, providing a unique and personalized experience for each user.

Delhi Eats is more than just a food guide - it's a community-driven platform that allows users to share their experiences, rate their favorite dishes, and connect with other food enthusiasts. With its robust features and engaging interface, Delhi Eats is the ultimate destination for anyone looking to explore the culinary delights of Delhi. Whether you're a seasoned foodie or just visiting the city, Delhi Eats is the perfect companion for your gastronomic adventures.

âœ¨ Features
==========

Here are some of the key features of Delhi Eats:

1. **Restaurant Directory**: A comprehensive list of street food restaurants and stalls in Delhi, complete with addresses, phone numbers, and reviews.
2. **Dish Recommendations**: Personalized recommendations for popular dishes and must-try items at each restaurant.
3. **User Reviews**: A platform for users to share their experiences and rate their favorite dishes and restaurants.
4. **Map View**: An interactive map that allows users to explore restaurants and food stalls in their vicinity.
5. **Search Functionality**: A robust search function that allows users to find specific restaurants, dishes, or cuisines.
6. **Food Blogs**: A section dedicated to food blogs and articles, featuring recipes, cooking tips, and stories from the culinary world.
7. **Community Forum**: A discussion forum where users can connect with other food enthusiasts, ask questions, and share their experiences.
8. **Notifications**: Personalized notifications for new restaurant openings, special deals, and events in the food scene.

ğŸ“ Project Structure
=====================

The project is organized into the following folders:

* `app`: Contains the main application code, including the Streamlit app and backend logic.
* `data`: Stores data related to restaurants, dishes, and user reviews.
* `images`: Holds images and other media files used in the application.
* `models`: Contains machine learning models used for recommendations and other features.
* `tests`: Includes unit tests and integration tests for the application.
* `utils`: Provides utility functions for tasks such as data processing and API calls.

==============

To run the application, follow these steps:

1. **Setup**: Install the required dependencies by running `pip install -r requirements.txt` in your terminal.

3. **Build**: Build the application by running `streamlit run app.py` in your terminal.
